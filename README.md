# React Investments

## Installation

This project requires [Node.js](https://nodejs.org/) v15+ to run.

Install the dependencies and devDependencies and start the server and frontend.
to run server:

```shdillinger
cd server
yarn i
yarn start
```

to run frontend:

```shdillinger
cd web
yarn i
yarn dev
```
